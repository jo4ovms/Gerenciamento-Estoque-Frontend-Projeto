# Front-End Gerenciamento de Estoque

# Como usar:

1. Criar o banco testdb_spring
2. Inserir no banco:
   INSERT INTO roles(name) VALUES('ROLE_USER');
   INSERT INTO roles(name) VALUES('ROLE_MODERATOR');
   INSERT INTO roles(name) VALUES('ROLE_ADMIN');
3. Rodar o backend
4. Rodar o front:
   - npm start
